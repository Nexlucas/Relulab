# ReluLab 💊

Landing page desenvolvida para a empresa Biolab, com foco em apresentar a marca, seus serviços e informações institucionais de forma moderna, responsiva e intuitiva.

## 📌 Sobre o projeto

O ReluLab é uma landing page criada para representar a identidade visual da Biolab, uma empresa do setor farmacêutico.  
O projeto foi desenvolvido com foco em:

- Layout moderno
- Responsividade
- Organização visual
- Boa experiência do usuário
- Estrutura limpa e reutilizável

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Google Fonts
